Game Description

Brangwen is a collaborative project with two other friends of mine over a weekend for the global game jam earlier this year. This game was made involving unity and C#, and the premise is to capture the difference between light and dark. It was a barebones survival shooter, fighting waves of zombies that attack you. The level was designed with the theme of dark and light in mind. The zombies can either be ‘dark’ or ‘light’, and only take damage when you stand in the part of the map that correlates to their theme.

Game Description

Oregon Trail is a web based game developed using javascript and built out on the web utilizing html/css. The game files are separated by client side and server side. It utilizes functioning APIs, and a database to record the highscore. The game itself functions like the classic ‘Oregon Trail’ game, where the player can decide how they want to go through the trail, with riskier but more rewarding options increasing the chance of death within their party, while safer options, poses risks of being too slow.

Space Arena

Game Description

Space Arena is a game developed in C++ with the use of SFML. It is a top down arena game with scaling levels and enemies, and allows the user to utilize powerups after each level to help them along the way. The player has to survive a wave of zombies each level, and after each level the arena gets bigger. There are also player drops such as ammo and health packs to help the player.
